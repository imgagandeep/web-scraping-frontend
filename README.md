# Web Scraping Frontend

In this project scrape a website home page and getting a these information:

1. Name
2. Description
3. Company Logo
4. Address
5. Phone Number
6. Email
7. Website URL
8. Website Screenshot
9. Social Profiles

    1. Facebook
    2. Instagram
    3. Linkedin
    4. Twitter
